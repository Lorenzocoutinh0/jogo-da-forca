# Jogo da Forca

Projeto desenvolvido em Python durante meus estudos de programação.

## Sobre o projeto

Este projeto implementa o clássico Jogo da Forca utilizando:

* Python
* Modularização de código
* Manipulação de arquivos
* SQLite para armazenamento de pontuação

O objetivo foi praticar a organização de projetos maiores, separação de responsabilidades em arquivos diferentes e persistência de dados.

## Estrutura do projeto

* `main.py` → ponto de entrada do programa.
* `jogo.py` → regras e fluxo principal do jogo.
* `desenhos.py` → funções responsáveis pela exibição da forca.
* `bd.py` → comunicação com o banco de dados SQLite.
* `palavras.txt` → lista de palavras utilizadas no sorteio.
* `score.db` → banco de dados com as pontuações.

## Conceitos praticados

* Funções
* Loops
* Condicionais
* Listas
* Strings
* Modularização
* Importação de módulos
* SQLite
* Manipulação de arquivos

## Como executar

```bash
python main.py
```
